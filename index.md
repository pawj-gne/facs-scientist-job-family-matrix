## How to Participate

### Read and Comment

You can highlight any text on the site to start a discussion about it:

1. **Select text** on any page (10-1000 characters)
2. A **"Discuss this"** button appears above your selection
3. Click it to open the forum with your quote included

### Use the Discussion Forum

The **[Discussion Forum]({{ '/forum' | relative_url }})** is where we collaborate. Use it to:

- Ask questions about the framework
- Suggest changes or additions
- Share feedback on draft documents
- Discuss ideas with colleagues

**Note:** The forum requires a GitHub account. If you haven't set it up yet, see the [Forum Setup Guide]({{ '/FORUM-GUIDE' | relative_url }}).

### Submit Documents

Have a document to contribute? Use the **[Submit Document]({{ '/converter' | relative_url }})** tool to convert Word docs to the site format.
